# Projet L4INA01 - Programmation pour les Sciences du Langage
**Étudiante : Soukaina Elkasmi**

## Description
Ce projet a été réalisé dans le cadre du cours L4INA01. Il s'agit d'un outil d'analyse de données textuelles permettant de manipuler des fichiers au format `.txt`. 

L'application est divisée en deux parties :
1. **Un Curriculum Vitae** : Présentation de mon parcours et de mes lectures.
2. **Un Analyseur de texte** : Un outil capable de segmenter le texte, de générer un dictionnaire de fréquences et d'effectuer des recherches spécifiques.

## Fonctionnalités
L'analyseur propose les outils suivants :
- **Segmentation** : Découpage du texte selon des délimiteurs personnalisables (RegExp).
- **Dictionnaire** : Calcul de la fréquence d'apparition de chaque mot.
- **GREP** : Extraction des lignes contenant un pôle spécifique.
- **Concordancier** : Affichage du contexte gauche et droit d'un mot choisi.
- **Statistiques** : Comptage automatique du nombre de formes dans le texte.

## Structure du projet
Conformément aux consignes du cours, le projet est organisé ainsi :
- `index.html` : Structure de la page web.
- `assets/css/style.css` : Mise en forme et couleurs (Violet, Beige, Rose et Jaune).
- `assets/js/main.js` : Logique de traitement de texte en JavaScript.
- `assets/img/` : Images utilisées pour le CV.

## Utilisation
Pour tester l'outil :
1. Accédez au lien GitHub Pages.
2. Cliquez sur "Choisir un fichier" et sélectionnez un fichier `.txt`.
3. Utilisez les boutons d'analyse pour afficher les résultats dans l'encadré jaune.
